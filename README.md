# Split-the-Bill (EquiSplit)

Full-stack expense-sharing platform.

## Prerequisites

- Node.js 18+

## Local development

### Backend API (port 3000)

```bash
cd services/api
npm install
npm run dev
```

Health check: `GET http://localhost:3000/health`

### Frontend (port 5173)

```bash
cd apps/web
npm install
npm run dev
```
